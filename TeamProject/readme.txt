Readme for chess game.

To install:
	1. download zip and extract anywhere
	2. ensure that the computer hosting the server is running a mysql database labeled "student_space" accessible for username: "student" and password: "hello"
	3. in mysql, while using the student_space database, copy and paste the Database.sql contents included in the extracted files into your mysql terminal while using the database
	4. run the batch file chessServer.bat to start the server on the computer that will host the server.
	5. run the batch file chessClient.bat to start the game client.
	6. enjoy the game.
